# Entitlement To Receive Checker

Description: A Chrome Extension that can be used to search the GPHC or
the GMC register for needed data. Such as a doctor's registration
details. Or a Pharmacy's registration details.

## Main Features

- Popup extension that gives the user option to search either GPHC or
  GMC.
- Loads either GPHC or GMC depending on option given.
